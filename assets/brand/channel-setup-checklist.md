# YouTube Channel Setup Checklist — miniature-guacamole

Work through this top to bottom. Each section is a distinct YouTube Studio task.

---

## 1. Channel Identity

- [ ] Channel name: `miniature-guacamole`
- [ ] Handle: `@miniature-guacamole`
- [ ] Channel description: copy from `brand-guide.md` (YouTube Studio About field — the verbatim block under "Channel Description Copy")
- [ ] Channel icon: use the EM capybara (guac green, `#4A7C59`) or the `mg` wordmark — whichever renders cleanest at 800x800 and 98x98. Confirm at small size before uploading.

---

## 2. Channel Art

- [ ] Banner dimensions: **2560 x 1440 px** — YouTube requires this for full-bleed
- [ ] Safe zone (visible on all devices): **1546 x 423 px** centered — keep all text and logo within this area
- [ ] Banner content: "AI Dev Team for Claude Code" — Inter Bold, `#E8E8E8` on `#121820` background. List the 16 skills or the 6 character names as secondary text in `#8A9BB0`.
- [ ] Upload via: YouTube Studio > Customization > Branding > Banner image

---

## 3. Playlists

Create these 4 playlists before publishing Episode 1. Add episodes as they publish.

- [ ] **Getting Started** — Episodes 1, 2
  - Description: "Start here. What miniature-guacamole is and how to install it in 60 seconds."
- [ ] **Core Concepts** — Episodes 3, 5, 7
  - Description: "The CAD cycle, the agent org chart, and how memory works. The architecture behind the framework."
- [ ] **Tutorials** — Episodes 4, 6, 8, 9
  - Description: "Hands-on. Real projects, real skills, real output."
- [ ] **Season 1** — Episodes 1–10 (full season playlist)
  - Description: "All 10 episodes from Season 1."

Playlist order: set each playlist to manual order and sequence episodes in episode order.

---

## 4. End Screen Template

- [ ] Duration: last **20 seconds** of every video
- [ ] Elements:
  - Watch-next card (right side) — YouTube auto-picks or manually set to next episode
  - Subscribe button (left side)
- [ ] Set this on Episode 1 before publishing, then copy the end screen to each subsequent video
- [ ] Do not add a "like" button overlay — keep it to 2 elements max

---

## 5. Pinned Video

- [ ] Pinned video: **Episode 2 — "Install in 60 Seconds"**
- [ ] Rationale: Episode 1 drives intent, Episode 2 converts it. New visitors who land on the channel page should see the install tutorial first.
- [ ] Set once Episode 2 is published and live — not before

---

Complete these fields in YouTube Studio > Customization > Basic info:

- [ ] Description: (same as channel description — see step 1)
- [ ] Links to add:
  - [ ] GitHub: `https://github.com/rivermark-research/miniature-guacamole`
  - [ ] Discord: [add invite link when available]
- [ ] Business email: optional — add if you want to surface a contact email

---

## 7. Featured Channels

- [ ] Add Anthropic's official YouTube channel as a featured channel
- [ ] Add the Claude Code official channel if one exists
- [ ] Location: YouTube Studio > Customization > Featured channels

---

## 8. Community Tab

- [ ] Community tab: **do not enable yet**
- [ ] Enable once the channel has 500+ subscribers (YouTube unlocks this anyway at that threshold on some accounts)
- [ ] First community post: Discord invite with a short "come hang out" message — not a sales pitch
- [ ] Use community tab for: Discord drives, episode announcements, community questions. Not for: automated reposts or promotional content.

---

## 9. Upload Defaults

Set these in YouTube Studio > Settings > Upload defaults. Saves time per video.

- [ ] Visibility: Scheduled (not auto-public)
- [ ] License: Creative Commons (Attribution)
- [ ] Category: Science & Technology
- [ ] Comments: Enabled, held for review (reduce spam without turning them off)
- [ ] Captions: Auto-generated (no manual upload required)

---

## 10. Channel Verification and Permissions

- [ ] Verify the channel (phone verification) — required to upload thumbnails
- [ ] Custom URL: once eligible (100+ subscribers), claim `youtube.com/@miniature-guacamole`
- [ ] Two-factor auth: enabled on the Google account that owns the channel

---

## Launch Gate

Only publish Episode 1 when all of the following are true:

- [ ] Channel identity complete (steps 1–2)
- [ ] Playlists created (step 3)
- [ ] End screen template ready (step 4)
- [ ] Episode 2 is produced and scheduled (not yet published)
- [ ] Thumbnail for Episode 1 uploaded
- [ ] Channel verified
