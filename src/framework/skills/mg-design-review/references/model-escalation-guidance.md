## Model Escalation

This skill runs on Sonnet for cost efficiency. Follow `../_shared/model-escalation.md` protocol.

**Escalate to Opus-tier agent (ceo) when:**
- Design changes affect brand identity or market positioning
- Conflicting design direction between art-director and product requirements
- Design system changes that impact all future components

**Stay on Sonnet for:**
- Brand consistency checklist (colors, typography, spacing)
- Responsive breakpoint validation
- Coordinating art-director and design specialists
- Structured UX pattern evaluation
